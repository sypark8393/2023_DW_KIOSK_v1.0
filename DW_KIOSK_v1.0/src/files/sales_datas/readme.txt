***** CSV 파일 한글 데이터 깨짐 문제 해결 방법 *****
입력하신 경로에 생성된 CSV 파일은 인코딩 문제로 한글 데이터가 정상적으로 확인되지 않을 수 있습니다.
아래 방법에 따라 인코딩 형식을 변환하신 후 사용하시기 바랍니다.

1) 생성된 CSV 파일을 마우스 오른쪽 버튼으로 클릭
2) [연결 프로그램]을 클릭한 후 [메모장] 선택
3) 메모장 상단의 [파일] 메뉴에서 [다른 이름으로 저장] 클릭
4) 팝업창 하단의 [파일 형식] 항목에서 드롭다운 메뉴를 펼치고 "모든 파일" 선택
5) 팝업창 하단위 [인코딩] 항목에서 드롭다운 메뉴룰 펼치고 "ANSI" 선택
6) 원래 CSV 파일을 더블 클릭
7) 이미 있는 파일을 바꾸겠냐는 창이 뜨면 [예] 클릭
8) 메모장을 닫고 CSV 파일을 엑셀에서 열기

※참고 사항※
1.
대기번호, 거래일시, 회원 ID의 경우 엑셀에서 확인할 때의 내용과 콘솔 창에서의 출력 내용에 차이가 있을 수 있습니다.
엑셀에서 숫자값을 처리하는 방식으로 인해 발생하는 사항이므로, 시스템과는 무관한 점 참고 바랍니다.

2.
쉼표(,)가 포함된 데이터의 경우 CSV 파일로 변환하며 열의 위치가 밀릴 수 있습니다.
CSV 파일에서 쉼표를 구분자로 사용하기 때문에 발생하는 문제이므로,
중요한 데이터의 경우 콘솔 창에서 출력 및 조회하여 확인하는 것을 권장드립니다.
